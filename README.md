*Garuda Island* is a premium, beautifully crafted desktop overlay inspired by modern mobile UI aesthetics but built entirely for Windows 11. It seamlessly integrates into your workflow, providing instant access to vital system stats, smart productivity tools, and a centralized control center—all wrapped in a gorgeous, fully customizable UI.

## ✨ Key Features

### 🎨 Stunning Visual Design
- **Native Windows 11 Acrylic/Mica Blur:** Enjoy true deep integration with the OS styling.
- **Dynamic Theming:** Seamless Dark/Light mode matching your system.
- **Seamless Animations:** Smooth hover effects, transitions, and pill-expansion physics.

### 🔋 Advanced System Statistics
- **Smart Battery Tracker:** Get instant visual alerts for charging state, critical battery levels, and time remaining.
- **Network Monitor:** Live WiFi & Bluetooth status at a glance.
- **Audio Integration:** Instant system volume control through the dynamic transient pill.

### 🌤️ Live Weather & Context
- **Smart Location:** Auto-detects your city to give real-time weather updates right on your screen.
- **Currency Converter API:** Instantly track global exchange rates.
- **Time & Date:** Elegant clock display that never gets in your way.

### 🚀 Productivity Powerhouse
- **Focus Mode:** Blocks out distractions with customizable timers.
- **Multi-Clipboard Manager:** Saves your copy history natively so you never lose a link or text snippet again.
- **Global Hotkeys:** Press `Ctrl+Alt+G` anytime to instantly summon the Island over any fullscreen app!

### ⚙️ Seamless Windows Integration
- **Zero-Friction Startup:** Native boot-launch integration means the Island is there the moment you log in.
- **Always on Top (Optional):** Keep the pill floating securely above all your other windows.
- **Professional Installer:** Installed securely via an Inno Setup built `.exe` without requiring permanent admin rights for daily usage.
- **Kill-Switch Security:** Built-in OTA version verification ensures users always stay on supported and secure builds.

## 📥 Installation

1. Download the latest installer from the [Releases](https://github.com/YOUR_GITHUB_USERNAME/GarudaIsland/releases) section.
2. Run `GarudaIsland_v1.0.0_Setup.exe`.
3. Follow the setup wizard (You can enable Auto-Start and Premium Blur here).
4. Enjoy your new Island! Press `Ctrl+Alt+G` to toggle the Control Center anytime.

---

## 🔒 Version Control & Security

This application features a remote security check. To continue using Garuda Island, please ensure your system is connected to the internet periodically to verify your license version. Outdated or deprecated versions will automatically prompt for an update.
